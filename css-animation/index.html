Accessibility

Animation on the web should be used responsibly. Consider those with seizures and other disorders and do not create animations that can be harmful. See WCAG 2.1 Guideline 2.3 Seizures and Physical Reactions for more details.

Developers should now be using prefers-reduced-motion where needed, which can be helpful for those users who disable animations by default, on their operating system of choice. You can learn more about this media query on MDN Web Docs.
