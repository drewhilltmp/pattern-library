This pattern will allow you to easily place and control the presentation of an accordion user interface (UI) .

Dependencies

  • HTML - HTML for all variations are the same.
  • CSS - The CSS is written in Sass, so you will need to install it. Once installed, command line to the directory where init.scss lives and run: sass --watch init.scss:init.css --style compressed
  • JavaScript - jQuery is required to function.

Download Repository - Please note that Jekyll is a requirement to run the repository locally. Contact Michael Spellacy (Spell) for further setup instructions.
